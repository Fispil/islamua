# Islam UA Prayer App

## Overview
A simple prayer application designed to help users manage and track their prayer requests and responses.

## Features
- Create and organize prayer requests
- Track prayer history
- Set reminders for prayer times
- View answered prayers

## Getting Started
1. Clone the repository
2. Install dependencies
3. Run the application

## Usage
Launch the app and begin adding your prayer requests. Mark prayers as answered when resolved.
